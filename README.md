# Recetario-comun

Ejercicio: Recetario Colaborativo
Este repositorio es una actividad práctica para aprender a utilizar el flujo de trabajo con Forks en GitHub.

Escenario
Eres un colaborador que quiere añadir su receta favorita a este proyecto de código abierto. Como no tienes permisos de edición directa, debes seguir los pasos detallados a continuación.
